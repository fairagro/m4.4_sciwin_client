## Get started
<Steps>
1. ### Install SciWIn-Client
	The easiest way to install `s4n` is to use the installer scripts.
	<h4 class="flex items-center gap-2"><Icon name="linux" label="Linux"/> Linux</h4>
	```bash 
	curl --proto '=https' --tlsv1.2 -LsSf https://fairagro.github.io/sciwin/get_s4n.sh | sh 
	```
	<h4 class="flex items-center gap-2"><Icon name="seti:windows" label="Windows"/> Windows</h4>
	```powershell
	powershell -ExecutionPolicy Bypass -c "irm https://fairagro.github.io/sciwin/get_s4n.ps1 | iex"
	```
2. ### Initialize Project
	Initialize a SciWIn Project in your code repository
	```bash
	s4n init
	```
3. ### Start creating Workflows
	Using the `s4n tool` and `s4n workflow` commands you can built your reusable data pipelines stepwise.
	```bash
	s4n tool create echo "Hello World"
	```
</Steps>
