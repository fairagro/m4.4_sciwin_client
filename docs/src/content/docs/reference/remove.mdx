## `remove`
The `remove` or `rm` command can be used to delete a CWL File while checking whether its linked in another one. 

    ```txt frame="terminal" title="Usage"
    Removes a CWL File from the workflows Directory

Usage: s4n remove <FILE>

Arguments:
  <FILE>  

Options:
  -h, --help  Print help
    ```
