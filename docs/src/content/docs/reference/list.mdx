## `list`

`list` or `ls` can be used to list all existing CWL Workflows and Tools. Using the command without the `-a` flag just ouputs the names of all existing CWL files in the project. Using the `-all` (or `-a`) flag will also output the tools in- and outputs which than can easily be used for the `connect` command.

    ```txt frame="terminal" title="Usage"
    Lists either all CWL Files or details to a given file

    Usage: s4n list [OPTIONS] [FILE]

    Arguments:
      [FILE]  

    Options:
      -a, --all   Outputs the tools with inputs and outputs
      -h, --help  Print help
    ```
